## Managing Contacts

### Adding New Contacts

<Tabs>
  <Tab title="Manual Entry" icon="plus">
    Create contacts manually by entering their information.
    
    <Steps>
      <Step title="Tap Add Button" icon="plus-circle">
        Tap the plus icon in the contacts tab.
      </Step>
      <Step title="Enter Details" icon="edit">
        Fill in name, phone, email, and other information.
      </Step>
      <Step title="Add Photo" icon="camera">
        Upload a photo or use the camera to take one.
      </Step>
      <Step title="Save Contact" icon="save">
        Tap save to add the contact to your database.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Import Options" icon="upload">
    Import contacts from your device, email, or social media.
    
    <CodeGroup tabs="iOS,Android">
      ```swift
// iOS Import Example
import Contacts

let store = CNContactStore()
store.requestAccess(for: .contacts) { granted, error in
    if granted {
        // Import contacts logic
    }
}
      ```
      ```kotlin
// Android Import Example
import android.provider.ContactsContract

val cursor = contentResolver.query(
    ContactsContract.Contacts.CONTENT_URI,
    null, null, null, null
)

// Process cursor for contact data
      ```
    </CodeGroup>
  </Tab>
</Tabs>

## Communication Integration

Lazaar Haythem integrates with your favorite communication apps to provide a unified contact experience.

<Steps>
  <Step title="Link Accounts" icon="link">
    Connect your email and messaging accounts in settings.
  </Step>
  <Step title="Sync Data" icon="refresh-cw">
    Allow the app to sync contact information from linked accounts.
  </Step>
  <Step title="View History" icon="clock">
    Access communication history directly from contact profiles.
  </Step>
</Steps>

<Callout kind="info">Integration requires granting appropriate permissions to ensure secure data access.</Callout>

## Troubleshooting

<Expandable title="Common Issues" default-open="true">
  | Issue | Solution |
  |-------|----------|
  | Contacts not syncing | Check account permissions and internet connection |
  | Duplicate contacts | Use the merge feature in the contacts tab |
  | Search not working | Ensure search index is up to date in settings |
  | App crashing | Clear app cache or reinstall the latest version |
</Expandable>

<Expandable title="Getting Help" default-open="false">
  If you encounter persistent issues, contact our support team through the app's help section.
